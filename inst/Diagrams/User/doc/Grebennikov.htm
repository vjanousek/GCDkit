GrebennikovR Documentation

Grebennikov (2014) 5Fe2O3t - Na2O+K2O - 5(CaO+MgO)

Description

Assigns data for 5Fe2O3t - Na2O+K2O - 5(CaO+MgO) ternary diagram into Figaro template (list 'sheet') and appropriate values into 'x.data' and 'y.data'.

Usage

Grebennikov()

Arguments

None.

Details

The ternary plot 5Fe2O3t - Na2O+K2O - 5(CaO+MgO) of Grebennikov (2014) serves for classification of A-type granites and related felsic volcanic rocks (SiO2 > 67 wt.%).

Value

sheet

list with Figaro Style Sheet data

x.data

x coordinates

y.data

y coordinates

Author(s)

Vojtech Janousek, vojtech.janousek@geology.cz

References

Grebennikov AV (2014) A-type granites and related rocks: petrogenesis and classification. Russ Geol Geophys 55: 1353-1366 doi: 10.1016/j.rgg.2014.08.003
